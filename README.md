# Simple Site - Al-Qamar College

A React.js project for **Al-Qamar College** website.

## Features

- Responsive Navbar with scroll effect
- Home section with background image
- About, Admissions, Programs, and Contact sections
- Conditional Rendering in Admissions
- Dynamic Programs list
- Footer

## Technologies Used

- React.js
- Tailwind CSS
- Vite.js

## Learned in this Project

- React Functional Components
- Props
- useState (state management)
- useEffect (side effects)
- Conditional Rendering
- List Rendering
- Tailwind CSS styling
- Dynamic class handling
- Using local assets (images)
- Project structuring
